## Copilot Coding Agent

This repository includes a GitHub Copilot coding agent scaffold. The agent workflow is located at `.github/workflows/copilot-coding-agent.yml` and can be triggered manually or on a schedule. To enable the agent, set the repository secret `AGENT_ENABLED` to `true`.
